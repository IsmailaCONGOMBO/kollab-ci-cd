# Kollab Project

![CI/CD Pipeline](https://github.com/IsmailaCONGOMBO/kollab-ci-cd/workflows/Kollab%20CI/CD%20Pipeline/badge.svg)

Application de gestion collaborative avec Docker, Kubernetes et CI/CD.

## Architecture
- **Backend**: Laravel 12 (PHP 8.2)
- **Frontend**: Next.js 15 (React)
- **Base de données**: MySQL 8.0
- **Orchestration**: Kubernetes
- **CI/CD**: GitHub Actions

## Déploiement
- Docker Compose (développement)
- Kubernetes (production)
- Pipeline automatique sur push
